# 과제 개요
32 bit 숫자가 파일에 4바이트의 크기로 저장되어 있다(network byte order). 1개 이상의 파일로부터 숫자를 읽어 들여 그 합을 출력하는 프로그램을 작성하라.
## 상세
- 4바이트 정수를 처리하기 위해서 `uint32_t` 형식을 사용
- `fopen`, `fread`, `fclose` 함수 등을 사용
- 정수 덧셈에서 발생하는 overflow는 무시
- 파일이 제대로 열리지 않는 경우, 파일의 크기가 4바이트보다 작아서 제대로 읽지 못했다고 판단되는 경우 등에 대해 에러 처리

# 흐름
1. 파일 개수만큼 for문을 돌도록 한다.
2. `main`의 인자로 `argc`, `argv`을 통해 파일의 개수와 파일들을 받는다.
3. `fopen`을 통해 파일을 연다.
   - 파일을 여는 과정에서 문제가 발생했을 경우, 프로그램을 종료한다.
4. `fread`를 통해 파일을 읽는다.
   - 파일의 크기가 4바이트보다 작거나 읽는 과정에서 문제가 발생했을 경우, `fclose` 실행 후 프로그램을 종료한다.
5. 각 버퍼값을 shift 및 or 연산하여 리틀 엔디안 수를 만든다.
6. `ntohl` 함수를 이용해 리틀 엔디안을 빅 엔디안으로 변환한다.
7. 피연산자와 연산자를 출력한다.
8. 덧셈을 수행한다.
9. `fclose`로 파일을 닫는다.
10. for문이 종료되면 덧셈 결과를 출력한다.
